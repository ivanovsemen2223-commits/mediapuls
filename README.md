# Медиапульс — лендинг диджитал-агентства

Одностраничный сайт для диджитал-агентства полного цикла: услуги, ценности, команда, отзывы клиентов и форма заявки.

**Живая версия:** https://mediapulse-delta.vercel.app

## Что показывает этот кейс

- **Швейцарская типографика** — строгая чёрно-белая сетка, гигантский заголовок-логотип, акцентные рамки-карточки.
- **Структура продающего лендинга** — оффер → ценности агентства → услуги с нумерацией → команда → отзывы → CTA-форма «Обсудим ваш проект?».
- **Аккуратный подвал** — логотип, описание, контакты (почта + Telegram).
- Полностью адаптивная вёрстка без фреймворков.

## Технологии

- HTML / CSS / vanilla JS — один файл, без сборки
- Шрифты: Inter / Inter Tight
- Хостинг — Vercel

## Как запустить

Статический сайт, сборка не нужна:

```bash
python3 -m http.server 8080
# открыть http://localhost:8080
```

Или просто открыть `index.html` в браузере.

## Структура

```
index.html   — вся страница (разметка + стили + скрипты)
assets/      — изображения (хиро, команда)
```
